== Description ==

Show the list of posts that was ordered by views count or comment count with useful color background for each post title. You can choose colors by yoursefl with color picker, pick posttype and number posts per page.

== Installation ==


1. Upload `rb-moist-views-colorful` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Appearence/Widget` menu then drag-drop `Rb Most Views` widget to your sidebar
